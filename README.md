# GST Profit Calculator 📊

A modern, single-page profit calculator that helps you calculate profit after buying from a vendor and selling to a customer — with full GST (Goods and Services Tax) support.

## 🚀 Features

- **Buy & Sell Tracking** — Enter purchase price from vendor and selling price to customer
- **GST Toggle** — Choose whether your entered price is *with GST* or *without GST*; the calculator handles the rest
- **Quick GST Rate Buttons** — One-click selection for 0%, 5%, 12%, 18%, and 28% GST rates (or enter any custom rate)
- **Profit After GST Liability** — See your exact take-home profit after settling GST with the government
- **Cash Flow Summary** — Visual cards showing what you pay, what you receive, and your GST liability
- **Detailed Breakdown** — Full line-by-line breakdown of purchase, sale, GST settlement, and final profit
- **Input Tax Credit (ITC)** — Automatically calculates GST input credit on purchases
- **Responsive Design** — Works beautifully on desktop, tablet, and mobile

## 📦 Getting Started

### Option 1: Open Directly

Simply open the `index.html` file in any web browser:

```bash
# Clone the repo
git clone https://github.com/rahulmasal/GST.git

# Open in browser
open index.html
```

### Option 2: GitHub Pages (Live URL)

The site is hosted via GitHub Pages. To set it up on your fork:

1. Go to your repo on GitHub → **Settings** → **Pages**
2. Under **Source**, select **Deploy from a branch**
3. Set **Branch** to `main` and folder to `/ (root)`
4. Click **Save**

Your site will be live at: `https://rahulmasal.github.io/GST/`

## 📱 How to Use

1. **Purchase Card** — Enter the price you pay your vendor (toggle between incl./excl. GST)
2. **Selling Card** — Enter the price you charge your customer (toggle between incl./excl. GST)
3. **GST Settings** — Select the applicable GST rate or enter a custom one
4. **View Results** — See profit after GST, cash flow summary, and detailed breakdown instantly

## 📋 Example

| Input | Value |
|-------|-------|
| Purchase Price (excl. GST) | ₹100 |
